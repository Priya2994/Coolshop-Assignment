## Tech Stack
**Client:** ReactJs, HTML, CSS

## Run Locally

Clone the project

```
  git clone https://github.com/Priya2994/Coolshop-Assignment.git
```

Go to the project directory

```
  cd Coolshop-Assignment
```

Install dependencies

```
  npm install
```

Start the server

```
  npm run dev
```

Compiles and minifies for production
```
npm run build
```
